
# Scientific Calculator

Scientific Calculator for performing basic and advanced mathematical operations.

## Features

* Basic arithmetic: `+`, `-`, `*`, `/`
* Trigonometric functions: `sin`, `cos`, `tan`
* Inverse trigonometric functions: `asin`, `acos`, `atan`
* Logarithms: `log`, `ln`
* Exponentials and powers: `x^y`, `e^x`
* Square root and factorial

## Usage

* Input numbers and select operations.
* Example calculations:

  ```
  2 + 3 = 5
  sin(30) = 0.5
  log(100) = 2
  sqrt(16) = 4
  ```
